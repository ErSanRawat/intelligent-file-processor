# Intelligent File Processor (Maven skeleton)


Production-ready skeleton focusing on concurrency primitives for a file processing pipeline in the automobile domain.


## Quick start
1. Ensure JDK 25+, Maven, Docker, Git are installed.
2. Build: `./mvnw -DskipTests package`
3. Run: `java -jar target/intelligent-file-processor-0.0.1-SNAPSHOT.jar`
